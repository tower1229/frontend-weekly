## 前端开发

- [基于原生 ES Module (ESM) 的现代打包工具 Snowpack](https://www.snowpack.dev/)

> 前端打包工具的未来一定是基于浏览器包管理功能的，对于多数前端来说不需要关注细节，你常用的 CLI 会默默的无缝切换到该技术，感受就是编译时间缩短，CPU 占用应该也会减小，因为浏览器在系统资源调度上有优势，性能不可能比 Nodejs 差。
