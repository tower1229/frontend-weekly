前端开发

  • 利用 Promise 的 cache 特性实现前端缓存

这应该是最简单的前端请求缓存实现思路

  • 项目推荐：@tower1229/axios

Axios的升级替代品，与Axios完全相同的API，仅仅为Instance.get()方法扩展缓存功能

  • 项目推荐：@tower1229/vue-store

更简单的vue状态管理插件，如果你也觉得vuex有点复杂，那么你需要这个

  • 密码强度检测算法

相对严谨的密码强度检测算法，对应的js实现

  • QGIS添加字段字段赋值

QGIS是个好东西，比Arcgis轻便很多，常用功能也都有

  • Git 通过submodule添加子项目/库(git仓库嵌套)

Git仓库嵌套真是个神奇的功能，特定情况下非常实用

  • UI自动化测试神器->Cypress

有空研究一下

  • Java代码获取图片格式及颜色模型

通过图片转16进制后图片内容特征进行识别。附：图片转16进制的前端实现

  • Blender+GoogleMaps+GIS插件生成三维地形图

Blender生成三维地形图
